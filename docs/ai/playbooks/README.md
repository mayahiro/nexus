# Nexus AI Playbooks

Playbooks are task-oriented operational documents for AI agents.

Use them after reading the main guide.

## Available Playbooks

- Migration projects: [docs/ai/playbooks/migration.md](migration.md)

## How To Use Playbooks

- open a playbook when the task already fits a known pattern
- use guides for command behavior and decision rules
- use playbooks for execution order and triage order
